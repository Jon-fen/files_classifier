# Clasificador de Documentos — Hospital Guillermo Grant Benavente

Aplicación web para clasificar y renombrar PDFs administrativos usando IA (Anthropic Claude Haiku).

Funciona con documentos escaneados desde papel, mejores resultados escaneos limpios y en buena calidad. Escaneos escritos a mano el resultado es variable dependiendo que tan legible sea la letra.
